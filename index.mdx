# Welcome to Flowxi

## Introduction

Welcome to **Flowxi**, the banking-grade API to build modern fintech products.
Flowxi lets you onboard users, manage wallets, move money, and stay compliant â€” with security and localization built-in.

This documentation helps you integrate fast, avoid mistakes, and ship production-ready fintech flows.

## What you can build with Flowxi

* **User onboarding**  
  Register users, verify email, enable 2FA, and manage lifecycle.
* **Wallets & balances**  
  Provision wallets, track balances, and secure funds.
* **Payments & transfers**  
  Move money securely with full auditability.
* **Compliance-ready flows**  
  KYC, KYB, and regulatory primitives built-in.
* **Localized APIs**  
  Errors, emails, and responses in user language (FR / EN fallback).
